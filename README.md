# 🚀 Python Basics – 140 Codes

This repository contains a collection of beginner-to-intermediate level Python programs focused on building strong fundamentals in programming and problem-solving.

🔗 GitHub Repo: https://github.com/deva190796/140-codes

## 📌 Overview

The goal of this repository is to:
- Strengthen core Python concepts
- Practice logical thinking and problem-solving
- Build consistency through hands-on coding

The programs cover a wide range of foundational topics including control flow, loops, mathematical computations, and basic algorithms.
